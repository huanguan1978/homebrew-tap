### homebrew-tap

#### Gai

Generative-AI Tools For Beginner

```shell
# uninstall
# brew untap huanguan1978/tap
# brew uninstall huanguan1978/tap/gai

# install
brew tap huanguan1978/tap
brew install --cask huanguan1978/tap/gai
```
